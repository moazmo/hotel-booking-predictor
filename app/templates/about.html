About This Project

The Hotel Booking Predictor is a machine learning application that predicts whether a hotel booking will be canceled or not based on various booking characteristics.

This application uses advanced machine learning techniques to analyze booking patterns and provide accurate predictions that can help hotel managers make informed decisions about their reservations and resource allocation.

Methodology
Data Preprocessing
  • Data Cleaning: Handled missing values and removed duplicates
  • Outlier Detection: Used IQR method to identify and remove outliers
  • Feature Engineering: Created new features like total_nights, total_guests, price_per_night
  • Multicollinearity: Removed highly correlated features to improve model performance
Model Training
  • Algorithms Tested: Logistic Regression, Random Forest, Gradient Boosting, SVM
  • Cross-Validation: Used stratified train-test split to maintain class balance
  • Feature Scaling: Applied StandardScaler to normalize numerical features
  • Categorical Encoding: Used one-hot encoding for categorical variables
Model Selection

The best performing model was selected based on accuracy score on the test set. The model also provides prediction probabilities for confidence scoring.
